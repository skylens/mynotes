# Shell 脚本

### 使用 `wget` `curl` 来执行 shell 脚本

```shell
$ wget -qO - http://skylens.github.io/echo.sh | sh
$ curl -sSL http://skylens.github.io/echo.sh | sh
```

### 使用短的 url 

- [git.io](https://git.io/)
- [ow.ly](http://ow.ly/url/shorten-url)
- [bit.ly](https://bitly.com/)

```shell
$ curl -sSL https://git.io/vddVc | sh
```
